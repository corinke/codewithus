About us

Code With Us is a website created by Corin and Maria as part of a project for Code First: Girls.

As we were learning to code websites, we'd spend hours scourering the internet for coding resources and courses which were accessible for learners. We decided to create a website which hosts 3rd party links to all of the resources and courses we've found useful throughout our time at Code First: Girls. By giving them a rating on skill level and content, we hope to save you some time searching and more time learning how to code!
